## 🛠️ Installation :

### *Dans le projet :*

- #### *Dépendances :*

  ```composer install```

  ```npm install```

  ```npm run build```

- #### *Base de donnée :*

    - créer la BDD
      `
      php bin/console doctrine:database:create
      `
    - créer les tables/colonnes
      `
      php bin/console doctrine:schema:create
      `
    - charger les données si il y en a
      `
      php bin/console doctrine:fixtures:load
      `
